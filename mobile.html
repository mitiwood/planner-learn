🦙 데모 모드 — Ollama 연결 가이드
1. ollama.com 에서 Ollama 설치 (Mac/Windows/Linux)
2. 터미널에서 ollama serve 실행
3. 모델 설치: ollama pull llama3 또는 ollama pull mistral
4. 위 URL에 http://localhost:11434 입력 후 연결 클릭
5. 🟢 연결됨 상태 확인 → 실제 AI 스트리밍 응답 사용 가능!
